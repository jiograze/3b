# Changelog

Tüm önemli değişiklikler bu dosyada belgelenecektir.

Format [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) standardını takip etmektedir,
ve bu proje [Semantic Versioning](https://semver.org/spec/v2.0.0.html) kullanmaktadır.

## [Yayınlanmamış]

### Eklenenler
- Service Mesh (Istio) entegrasyonu
- CI/CD pipeline (ArgoCD)
- Disaster Recovery sistemi
- Maliyet optimizasyonu araçları

### Değişenler
- Proje yapısı modernize edildi
- Dokümantasyon GitHub formatına uyarlandı
- Python paket yönetimi pyproject.toml'a taşındı

### Düzeltilenler
- Gereksiz dosyalar temizlendi
- Virtual environment yapılandırması düzeltildi
- Bağımlılık yönetimi iyileştirildi

## [0.1.0] - 2024-01-01

### Eklenenler
- Text-to-3D model dönüşümü
- Image-to-3D model dönüşümü
- Web arayüzü
- API endpoints
- Temel dokümantasyon
- Test altyapısı

### Değişenler
- Point-E modeli entegre edildi
- PyTorch3D entegrasyonu tamamlandı
- Veri işleme pipeline'ı oluşturuldu

### Düzeltilenler
- GPU bellek optimizasyonu
- Model yükleme performansı
- Hata yakalama mekanizmaları

### Güvenlik
- JWT tabanlı kimlik doğrulama
- Rate limiting
- Input validasyonu
